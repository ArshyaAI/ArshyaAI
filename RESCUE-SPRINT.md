# Same-day rescue sprint

A fixed technical rescue block for urgent Shopify, Railway, deployment, debugging, or AI automation blockers.

## Best fit

- Shopify store, theme, checkout, app, or integration issues
- Railway deploy, runtime, database, volume, or environment-variable problems
- Production debugging where the fastest path is reproduction, logs, and a focused patch
- AI automation workflows that are stuck before launch
- Agency overflow work where the client needs a same-day technical unblock

## Format

- 2h diagnostic/rescue sprint
- 4h implementation sprint
- Written result summary after the sprint
- Clear scope before kickoff
- No secrets pasted into GitHub issues or emails

## Pricing bands

- 2h diagnostic/rescue sprint: CHF 300-500
- 4h implementation sprint: CHF 700-1,200
- Emergency agency/client outcome sprint: scoped individually

The exact quote depends on urgency, access constraints, and the outcome required.

Email `ashbahal2000@gmail.com` or open a private-safe intake issue using the rescue sprint issue template.
